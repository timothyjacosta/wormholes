# Wormholes Beta 301 — Desktop-only support

Wormholes Beta 301 is a desktop-browser application. It does not provide phone-specific navigation, touch-only controls, safe-area handling, or a separate mobile interface.

- Supported browser families: Chromium, Firefox, and Safari on desktop operating systems.
- Layout contract: the complete desktop interface remains available at every browser-window width.
- Narrow desktop windows use ordinary responsive wrapping and stacking so the document does not require horizontal scrolling.
- Navigation remains the same desktop tab navigation; it wraps when necessary rather than becoming a mobile navigation system.
- Browser, accessibility, storage, security, and release checks use desktop browser profiles.
